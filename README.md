## 📌 Project Overview
Analysed 1000-patient clinical trial dataset to
identify adverse event patterns, dropout trends
and site performance metrics using Python and SQL.

## 🎯 Problem Statement
Clinical data quality is critical in trials.
This project identifies AE patterns, data quality
issues and site performance gaps to support
clinical decision making.

## 📊 Dataset
- 1000 patients across 4 clinical trial sites
- Variables: Demographics, AE data, SAE flags
- Focus: Adverse events and dropout patterns

## 🔬 Methodology
1. Clinical Data Collection & Cleaning
2. SQL Query Development for Site Analysis
3. Adverse Event Pattern Analysis
4. Data Visualization & Reporting

## 📈 Key Findings
- Identified AE patterns across treatment groups
- Site performance evaluation using SQL queries
- Patient dropout trend analysis
- Adverse event severity distribution

## 🖼️ Visualizations
- AE Severity Distribution
- Site Performance Comparison
- Patient Dropout Trends
- Age Distribution Analysis

## 🛠️ Tech Stack
- Python, Pandas, NumPy
- SQL for data querying
- Matplotlib for visualization
- Google Colab
